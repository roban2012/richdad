《现金流量表》


《现金流量表》的推导过程引用 《寒武商业》（成若般）的部分内容。

做《现金流量表》的方法有两种。第一种是中国采用的直接法，目的是配合上市公司查帐；第二种是美国采用的间接法，目的是方便管理应用。根据不同的目的，选择的工具也不同。中国政府为了防止企业做假账，把现金流量表搞得复杂无比，没有几个人搞得懂。要想把账作平交差，要么老老实实记录，要么“凑”。凑数、凑数，把数字摆平就可以了。由于直接法作账技术过于复杂，现在都不再要求中小企业必须提供《现金流量表》了。

而美国人作假账的代价很高，还有专门的审计师、律师去帮利益相关人查假账，等官司打赢了才收钱。所以，美国的会计更多的是为管理者服务的。简单地说，美国的间接法《现金流量表》对于任何具有初中水平的人来说，花一个星期就能学会。我们学财会不是为了当会计，也不是为了玩花样，所以我们只讲美国的《现金流量表》。

在掌握财会术语后，就可以用《资产负债表》公式推导《现金流量表》了。

A = L + OE ( I )

并且把

A = CA + NNA

L = CL + NCL

OE = Stock + RE

代入( I )得到

(CA +NNA) = (CL + NCL) + (Stock + RE) (II)

再把下面的

CA = Cash + AR +Inv.

NNA = NA C Dep.

CL = AP + WOE + TO

NI = Profit = RE + Dividend

RE = Profit C Dividend

代入( II )得到

[(Cash + AR +Inv.) + (NA C Dep.)] = [(AP + WOE + TO) + NCL] + (Stock + Profit C Dividend)

整理出现金：

Cash = [(AP + WOE + TO) + NCL] + (Stock + Profit C Dividend) C (AR +Inv.) C(NA C Dep.)

根据现金流量表的格式要求，得出现金流公式如下：

Cash = Profit + Dep. C AR C Inv. + AP + WOE + TO C NA + NCL + Stock C Dividend (III)

财会术语

1、《资产负债表》；

2、《利润表》。
